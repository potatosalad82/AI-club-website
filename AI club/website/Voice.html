Future goals

Launch, July 7, 2018

Welcome to the grand opening of Potato Schools. This is a project from Potato Industries with the goal to teach others coding from scratch to computer learning algorithms. We will release one group of lessons every month or so eventually creating a curiculm that spans multiple diffrent coding languages as well as 3D modelling, photoshop, and more. If you enjoy our lessons please donate so that we can keep everything free.
